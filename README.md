#📊 Analisis Ventas Pizzería

📖 Descripción del proyecto
Este proyecto consiste en un dashboard interactivo para la gestión de una pizzería. Desarrollado mediante análisis y visualización de datos con gráficos dinámicos, permite controlar ventas, pedidos e ingresos desde una interfaz moderna. Su objetivo es optimizar la gestión, facilitar la toma de decisiones e identificar tendencias y patrones.

🗂️ Estructura del proyecto
├── README.md # Descripción del proyecto
├── results/ # Gráficos y archivos de resultados
├── data/ # Datos originales

🛠️ Instalación y Requisitos
Este proyecto usa Google Sheets

📊 Resultados y Conclusiones
- Identificamos que las pizzas de categoría 'Classic' y tamaño 'L' son las que generan más ingresos
- Los viernes son los días con más pedidos, del mismo modo que la franja horaria 'Tarde' es donde hay más demanda
- Todos los meses hay aproximadamente el mismo número de pedidos
- La pizza 'The Classic Deluxe Pizza' es la más popular. Por el contrario, 'The Brie Carre Pizza' es la que menos demanda tiene
